# New-batocera-switch

For Batocera V42&+ only
Multi language installer (FR and EN)
This version of the code requires a file system for userdata that supports symlinking (EXT4, BTRFS) and is for x86_64 only!!  
This version integrates work from Foclabroc, DreamerCG and Spirit.  It does not include the bios keys.  

## EASY INSTALL
From a terminal window, run the following:<br>
```curl -L bit.ly/foclabroc-switch-all | bash```  (all version)  
or  
```curl -L foclabroc.short.gy/switch-all | bash```  (all version)
